# 📊 Calculadora de Médias

Projeto simples em Java que calcula a média de notas de um aluno.

## 🧠 Funcionalidades
- Recebe 3 notas via terminal
- Calcula a média aritmética
- Exibe se o aluno foi aprovado ou reprovado
